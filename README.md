# ACTIVIDAD 2 - TALLER DE LENGUAJES (CDO) 2026

## Estructura del proyecto

En el directorio `src` se encuentran los archivos `.py` de cada uno de los ejercicios de la práctica.

En el directorio `notebooks` se encuentra el archivo `.ipynb` en el que se importan los ejercicios y se ejecutan las funciones.

## Requisitos

* Python 3.x
* Jupyter Notebook

##  Instalación de dependencias

```bash
pip install -r requirements.txt
```

##  Ejecución

1. Ubicarse en la carpeta raíz del proyecto

2. Ejecutar:

```bash
jupyter notebook
```

3. Abrir el archivo:

```
notebooks/prueba_ejercicios.ipynb
```

4. Ejecutar todas las celdas
   (Las últimas dos corresponden al ejercicio 10)
